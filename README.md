<h2>如何导入名字列表？</h2>
<p>名字列表编写格式为：
名字：性别：外语类型

</p>
<p>例举1：张三me

</p>
<p>解析：张三（姓名），m（性别，female、male的首字母），e（外语类型：english,japanese的首字母）

</p>
<p>例举2：小红fj

</p>
<p>解析：小红（姓名），f（性别），j（外语类型）
</p>
<p>之后启动程序点击“导入”按钮即可导入
